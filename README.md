# Daniel Pekevski DotFiles

## How to run

Simply run:

```
bash <(curl -s https://raw.githubusercontent.com/pekevski/dotfiles/main/init.sh)
```

This will internally install Homebrew, Git and clone this repo and run `install.sh`!
